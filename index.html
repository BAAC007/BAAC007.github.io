01 · Perfil

Sobre mí

Soy Bryan Ávila, estudiante de Desarrollo de Aplicaciones Web (DAW). Me dedico a construir aplicaciones completas: desde el diseño de la base de datos hasta la interfaz que ve el usuario final.

Mi proyecto de fin de curso fue una plataforma de gestión para un concesionario de coches, donde trabajé tanto la parte de administración (CRUD completo) como el área de usuario. Actualmente estoy ampliando mis conocimientos en Java para fortalecer mi perfil como desarrollador full-stack.

Me interesa especialmente resolver problemas reales con código bien estructurado, y seguir aprendiendo buenas prácticas de desarrollo y seguridad en cada proyecto que hago.
